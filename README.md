# ♻ Waste Classification System (MLOps Project)

## 🌍 Overview
This project is an end-to-end waste classification system that uses a deep learning model to classify waste images into categories like plastic, metal, glass, paper, etc.

The system is deployed as a full-stack application with a cloud-based backend and an interactive frontend.
